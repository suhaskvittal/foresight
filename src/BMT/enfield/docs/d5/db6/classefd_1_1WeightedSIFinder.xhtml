Detailed Description

template<typename T>
class efd::WeightedSIFinder< T >

Extends the SIFinder class for weighted graphs.

This class finds a graph matching M=(Vm, Em) of G=(Vg, Eg) and H=(Vh, Eh) such that Eh is weighted.

So, the idea is to try to maximize the value of the edges matched.
