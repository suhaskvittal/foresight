Detailed Description

Maps every quantum and classic bit (not register) to a number inside a vector.

It also keeps track of the qubits inside every gate declaration. Inside every scope, it maps the qubits existing inside them to an uint32_t number.

Note that if "qreg r[10];" declaration exists, then "r" is not a qbit, but "r[n]" is (where "n" is in "{0 .. 9}").
