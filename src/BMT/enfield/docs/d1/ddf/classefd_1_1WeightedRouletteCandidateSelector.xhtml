Detailed Description

Selects the candidates randomly, based on their cost.

It first calculates the weight $w_i$ of each element $i$:

After, it calculates the probability $p_i$ of each element, based on their weigth:

Then, from a random number $r$ in [0, 1], we choose the first element $e$ such that $r < {i = 0}^{e}{p_i}$.
