Detailed Description

Keep track of the dependencies of each qbit for the whole program, as well as the dependencies for every gate.

Each gate, as well as the whole program have one 'DepsVector' variable. The idea is to store a sequence of parallel dependencies. Here, parallel dependency is a dependency that can't be broken down (unless the gate is inlined).
