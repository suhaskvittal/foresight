Detailed Description

Brute force solution to the Token Swap Finder.

It first preprocesses the given graph, generating all permutations from one starting point. This is the expensive part of the process. It takes O(|Q|!).

At each query, it renames the qubits and finds the swaps needed to solve that query. Each query only takes O(|Q|).
