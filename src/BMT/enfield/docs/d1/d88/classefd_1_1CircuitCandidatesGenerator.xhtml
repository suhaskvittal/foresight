Detailed Description

Generates a vector with the nodes that it can reach using the CircuitGraph.

Starting from the InputNodes in the CircuitGraph, it returns the nodes that are reachable from them within one step. After signalizing the processment of one Node, it advances all qubits and cbits related to that Node.

Almost like a graph traversal, after topological ordering.
